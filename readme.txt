# 🚀 SellerGrid AI – Review Intelligence Dashboard

A modern, premium UI dashboard designed to help eCommerce brands analyze and manage thousands of customer reviews efficiently.

---

## 📌 Overview

SellerGrid AI is a front-end dashboard prototype that demonstrates how businesses can:

* Analyze large volumes of reviews
* Identify sentiment trends
* Detect negative feedback spikes
* Export filtered data for decision-making
* Simulate AI-powered customer responses

This project focuses on **UI/UX, product thinking, and analytics visualization**, rather than backend functionality.

---

## 🎯 Problem Statement

eCommerce brands receive thousands of reviews across platforms like Amazon and Flipkart.
Manually analyzing this data is inefficient and time-consuming.

This dashboard provides a **centralized interface** to:

* Filter reviews
* Understand sentiment
* Track trends
* Take action quickly

---

## ✨ Key Features

### 📊 Analytics Dashboard

* KPI cards (Total, Positive, Negative, Neutral)
* Compact sentiment chart with percentage insights
* Weekly trend analysis with growth metrics
* Platform-level insights

### 🔔 Smart Alert System

* Detects spikes in negative reviews
* Highlights critical issues instantly

### 💬 Review Management

* Filter by platform, rating, and sentiment
* Real-time search functionality
* Pagination + Load More
* Interactive review cards

### 📄 Review Modal

* Click any review to view full details
* Displays user info, rating, and content

### 🤖 AI Reply Generator

* Simulates automated responses to customer feedback

### 📤 Data Export

* Export filtered reviews as CSV
* Includes:

  * Name
  * Email
  * Platform
  * Rating
  * Review text

### 🎨 Premium UI/UX

* Glassmorphism design
* Smooth animations
* Responsive layout
* Clean visual hierarchy

## ⚙️ How to Run

1. Download or clone the repository
2. Open the folder
3. Run `index.html` in any browser

No installation required 🚀

---

## 📥 Export Feature

Users can export filtered data:

* Navigate to **Reviews section**
* Apply filters (platform, rating, sentiment)
* Click **"Download Filtered Data"**

A `.csv` file will be generated containing:

```
Name, Email, Platform, Rating, Review
```
